Que é gasolineiras.gal?

gasolineiras.gal é unha web que che permite saber, por concellos, os prezos dos combustíbeis nas diferentes gasolineiras da Galiza. Como ves é unha aplicación moi simple e, polo tanto, con moita marxe de mellora.

De onde saen os datos?

Os datos que amosa esta web son obtidos da API Rest que ofrece o Ministerio para a Transición Ecolóxica e o Reto Demográfico.

Código

O código desta web está publicado en GitHub. Está feita co framework para python Flask. Calquera persoa que queira colaborar é benvida!
